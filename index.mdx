## Key Features

<CardGroup cols={2}>
  <Card title="Multiple Auth Methods" icon="key">
    Support for AWS Builder ID, IAM Identity Center, and automatic Kiro CLI sync
  </Card>
  <Card title="Intelligent Rotation" icon="rotate">
    Automatic multi-account rotation prioritizing lowest quota usage
  </Card>
  <Card title="Auto Recovery" icon="shield-check">
    Exponential backoff for rate limits and automated token refresh
  </Card>
  <Card title="Claude Models" icon="brain">
    Full support for Claude Sonnet, Haiku, and Opus including thinking modes

  <Card title="Context Management" icon="scissors">
    Gradual context truncation to prevent error 400 during retries

  </Card>
</CardGroup>
